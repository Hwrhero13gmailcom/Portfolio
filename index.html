02 — Project

GVUM Bot

Grab VR Unofficial Marketplace Live

A fully custom Discord marketplace bot built from scratch for the Grab VR community. Powers the entire creator economy — guided listing flows, private hire threads, reputation tracking, badge systems, disputes, and more. Every feature designed and coordinated by me using Node.js and Discord.js v14.

10-step DM listing flows with progress bars
Modal-first hire & apply thread system
Star rating & reputation system
Ambassador, Verified & Booster tiers
Two-party job done confirmation
Persistent JSON storage across restarts
Dynamic cooldowns per role tier
Mod tools, blacklist & dispute system
Automated escrow & expiry reminders
Weekly digest & category subscriptions
Node.js Discord.js v14 REST API JSON Storage dotenv
